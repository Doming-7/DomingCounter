## 💡 What It Does

A simple interactive counter with three controls:

| Button | Action |
|--------|--------|
| ➕ `+` | Increments the counter by 1 |
| ➖ `−` | Decrements the counter by 1 |
| 🔄 `Reset` | Resets the counter back to 0 |
